# AI Story Narrator

AI Story Narrator is a Streamlit application that generates stories using AI and converts them into audio narration.

## Features

- Story generation using Hugging Face LLM
- Audio narration using Hugging Face Text-to-Speech
- LangGraph workflow
- Streamlit user interface
- Audio download option

## Tech Stack

- Python
- Streamlit
- LangGraph
- Hugging Face Inference API

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py

## Environment Variable

HF_TOKEN=your_huggingface_token
